# 第十五章 Etcd 專案

`etcd` 是 `CoreOS` 團隊發起的一個管理設定訊息和服務發現 (`Service Discovery`) 的專案，在這一章裡面，我們將基於 `etcd 3.5 系列`版本介紹該專案的目標，安裝和使用，以及實現的技術。

> **版本說明：** 本章範例基於 etcd 3.5 系列版本編寫。etcd 官方維護最新兩個次版本（當前為 3.5 和 3.6）。請訪問 [etcd 官方發布頁](https://github.com/etcd-io/etcd/releases) 獲取最新版本訊息。
