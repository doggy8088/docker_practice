## 11.2 安裝與解除安裝

`Compose` 是 Docker 官方的開源專案，負責實現對 Docker 容器叢集的快速編排。

當前的 Compose 以 `docker compose` 子指令的形式提供。Docker Desktop 在 macOS、Windows 和 Linux 上預設包含它；如果你已經在 Linux 上單獨安裝了 Docker Engine 和 Docker CLI，也可以再安裝 Compose CLI 外掛。

### 11.2.1 Linux

在 Linux 上，你可以透過 Docker 官方發布頁安裝 Compose CLI 外掛。把二進位檔案儲存到 `$DOCKER_CONFIG/cli-plugins/docker-compose`，並賦予執行許可權即可。

```bash
$ DOCKER_CONFIG=${DOCKER_CONFIG:-$HOME/.docker}
$ mkdir -p $DOCKER_CONFIG/cli-plugins
$ curl -SL https://github.com/docker/compose/releases/latest/download/docker-compose-linux-x86_64 -o $DOCKER_CONFIG/cli-plugins/docker-compose
$ chmod +x $DOCKER_CONFIG/cli-plugins/docker-compose
```

### 11.2.2 測試安裝

```bash
$ docker compose version
Docker Compose version v5.x
```

### 11.2.3 解除安裝

如果是二進位封裝方式安裝的，刪除二進位檔案即可。

```bash
$ rm $DOCKER_CONFIG/cli-plugins/docker-compose
```
