## 11.2 安裝與解除安裝

`Compose` 是 Docker 官方的開源專案，負責實現對 Docker 容器叢集的快速編排。

從 `v2` 版本開始，`Compose` 作為 `docker` 的子指令存在，例如 `docker compose up`。

`Compose` 支援 Linux、macOS、Windows 10 三大平台。

`Docker Desktop for Mac/Windows` 自帶 `docker compose` 指令。

Linux 系統請使用以下介紹的方法安裝。

### 11.2.1 Linux

在 Linux 上，你可以透過下載 Docker Compose CLI 外掛 (二進位檔案名為 `docker-compose`) 來安裝。

從[官方 GitHub Release](https://github.com/docker/compose/releases) 處直接下載編譯好的二進位檔案即可。

> **提示**：版本更新較快，請訪問上述連結獲取最新版本號，替換下方指令中的版本號。

例如，在 Linux 64 位系統上直接下載對應的二進位封裝 (以 v5.0.2 為例)。

```bash
$ DOCKER_CONFIG=${DOCKER_CONFIG:-$HOME/.docker}
$ mkdir -p $DOCKER_CONFIG/cli-plugins
$ curl -SL https://github.com/docker/compose/releases/download/v5.0.2/docker-compose-linux-x86_64 -o $DOCKER_CONFIG/cli-plugins/docker-compose
```

之後，執行

```bash
$ chmod +x $DOCKER_CONFIG/cli-plugins/docker-compose
```

### 11.2.2 測試安裝

```bash
$ docker compose version
Docker Compose version v5.0.2
```

### 11.2.3 bash 補全指令

```bash
$ curl -L https://raw.githubusercontent.com/docker/compose/v5.0.2/contrib/completion/bash/docker-compose | sudo tee /etc/bash_completion.d/docker-compose > /dev/null
```

### 11.2.4 解除安裝

如果是二進位封裝方式安裝的，刪除二進位檔案即可。

```bash
$ rm $DOCKER_CONFIG/cli-plugins/docker-compose
```
