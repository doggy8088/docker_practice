# 容器監控

容器化技術的普及使得應用部署變得更加靈活和高效，但也給監控帶來了新的挑戰。

在傳統架構中，我們通常關注主機的 CPU、記憶體、磁碟 IO 等指標。而在容器環境下，除了主機層面的監控，我們更關注容器級別的資源使用情況、服務的執行狀態以及編排系統的健康狀況。

## 常見的監控方案

目前主流的容器監控方案包括：

* **cAdvisor**: Google 開源的容器資源監控工具，Docker 原生支援。
* **Prometheus**: CNCF 畢業專案，雲原生時代最流行的監控系統。
* **Grafana**: 強大的視覺化平台，常與 Prometheus 配合使用。
* **ELK/EFK Stack**: 主要用於日誌管理，但也能提供一定的監控能力。

本章將重點介紹如何使用 Prometheus 和 Grafana 搭建一套完整的容器監控系統。
