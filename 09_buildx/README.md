# 第九章 Docker Buildx

Docker Buildx 是一個 docker CLI 外掛，其擴充套件了 docker 指令，支援 [Moby BuildKit](9.1_buildkit.md) 提供的功能。提供了與 docker build 相同的使用者體驗，並增加了許多新功能。

> Buildx 需要 Docker v19.03+。在較新版本中已更常用且功能更完整。

## 本章內容

本章將詳細介紹 Docker Buildx 的使用，包括：

* [使用 BuildKit 建立映象](9.1_buildkit.md)
* [使用 Buildx 建立映象](9.2_buildx.md)
* [建立多種系統架構支援的 Docker 映象](9.3_multi-arch-images.md)
