# 第五章操作容器

容器是 Docker 又一核心概念。

簡單的說，容器是獨立執行的一個或一組應用，以及它們的執行態環境。對應的，虛擬機可以理解為模擬執行的一整套作業系統 (提供了執行態環境和其他系統環境) 和跑在上面的應用。

本章將具體介紹如何來管理一個容器，包括建立、啟動和停止等。
