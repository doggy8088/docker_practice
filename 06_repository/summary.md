## 本章小結

本章介紹了 Docker Hub 的使用、私有倉庫的搭建以及 Nexus 3 等企業級方案。

| 功能 | 說明 |
|------|------|
| **官方映象** | 優先使用的基礎映象 |
| **拉取限制** | 匿名 100 次/6h，登入 200 次/6h |
| **安全** | 推薦開啟 2FA 並使用 Access Token |
| **自動化** | 支援 Webhooks 和自動建立 |
